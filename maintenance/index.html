Upload Your Dataset

Upload a CSV file containing machine data. The system will automatically detect the required columns.

Expected Columns (case-insensitive):

  • Type (categorical: L, M, H)
  • Air temperature [K] or similar (numeric)
  • Process temperature [K] or similar (numeric)
  • Rotational speed [rpm] or similar (numeric)
  • Torque [Nm] or similar (numeric)
  • Tool wear [min] or similar (numeric)
  • Machine failure or similar (binary: 0/1)

Note: UDI and Product ID columns are optional and will be ignored if present.
